## 3. INFORMATION THE SOFTWARE DOES NOT COLLECT

We want to be clear about what we do NOT do. The Software:

- Does **NOT** upload, transmit, or share your files with us

- Does **NOT** access usernames, passwords, or credentials found during scans

### 4.1 License Management Data

We do **not** store your full payment card details. Payment is processed entirely by our third-party payment provider (see Section 7).

### 4.2 Anonymous Telemetry (Optional - Opt-In Only)

If you enable telemetry in Settings > Privacy, we collect:

| Data | Example | Purpose |
|------|---------|---------|
| Software version | "1.0.0" | Compatibility and update planning |
| Operating system type | "Windows 11" | Platform support |
| Scan duration | "45 seconds" | Performance optimisation |
| File counts | "1,234 files scanned" | Feature prioritisation |
| Feature usage | "Export to PDF used" | Product development |
| Error codes | "ERR_TIMEOUT_001" | Bug identification and fixes |

**What telemetry does NOT include:**
- File names, file paths, or file contents
- Scan results or findings
- Personal data detected during scans
- Your IP address (telemetry is anonymised before transmission)

**You can disable telemetry at any time** in Settings > Privacy. Telemetry is disabled by default.

### 4.3 Crash Reports (Optional - Opt-In Only)

If you enable crash reporting in Settings > Privacy, we may receive:
- Stack traces and error messages (code-level debugging information)
- Software version and OS version at time of crash
- Memory usage at time of crash

Crash reports do **not** include file contents, scan results, or personal data.

### 4.4 Support Communications

When you contact support@pcriskpro.com:
- Your email address
- Any information you voluntarily include in your message
- Log files you choose to attach (we will ask before using diagnostic data)

### 4.5 Vulnerability Database Queries

The Software queries public vulnerability databases to check for known security issues in your installed applications. These queries transmit:
- Application names (e.g., "Google Chrome")
- Application versions (e.g., "120.0.6099.71")

These queries do **not** transmit any personal data, file names, scan results, or information about your system beyond the application names/versions being checked.

### 4.6 Website Data

Our website (www.pcriskpro.com) may collect:
- **Essential cookies:** Necessary for website functionality (cannot be disabled)
- **Analytics cookies:** Anonymous usage statistics via a privacy-respecting analytics service (can be disabled via cookie banner)

Our website does not use third-party advertising trackers.

## 10. YOUR RESPONSIBILITIES AS A DATA CONTROLLER

### 10.1 When You Scan Third-Party Data
If you use the Software to scan systems containing other people's personal data (e.g., scanning a company file server), **you are the Data Controller** under UK GDPR for that personal data. You must:

- Have a **lawful basis** for processing that personal data
- Conduct a **Data Protection Impact Assessment (DPIA)** if scanning is likely to result in a high risk to individuals
- Implement **appropriate security measures** for any findings
- **Honour data subject rights** (access, erasure, etc.) for individuals whose data is discovered
- **Report breaches** to the ICO within 72 hours if applicable
- Maintain a **Record of Processing Activities** as required

### 10.2 PCRiskPro's Role
PCRiskPro is a **tool** that runs on your device. We:
- Are **not** a Data Processor (as defined under UK GDPR) because we do not process personal data on your behalf
- Do **not** receive, access, or store any personal data discovered during your scans
- Cannot be held responsible for how you use scan results

If you require a Data Processing Agreement (DPA) for your compliance records, please contact legal@pcriskpro.com and we will clarify our role in writing.
